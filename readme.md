## 🏪 Carrera Superstore Sales
Perform Forecasting Total Sales on Sales Superstore Dataset

### 🕵️ Carrera Goal
* [x] Find Optimum Model to Perform 7 Days Forecast in Total Sales Dataset
* [x] Creating Dashboard on Google Data Studio

### 🧠 Model
* Our Daily Sales Dataset is Already Stationary. Resulting in *`d`* Value is 0
* Our Partial Autocorrelation Plot is Suggesting *`p`* Value is 1 or 2
* Our Autocorrelation Plot is Suggesting *`q`* Value is 1 or 2
* Out of Multiple Trials, Our Best Model is Seasonal Arima. Using (2, 0, 2) (1, 1, 0, 12) Order Combination.
* Seasonal Arima is Considerably Well Perform Due to Minimal Error Rate, Compare to Other Model
* Seasonal Arima is Match With Daily Total Sales Dataset is Due to High Seasonality Pattern in Our Dataset
